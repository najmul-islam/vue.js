## Setup

To get started with the projects in this repository, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/najmul-islam/vue.js.git
   cd vue.js
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run serve
   ```
